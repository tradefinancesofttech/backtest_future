# Backtest Future Project

## Overview

The Backtest Future project aims to provide a robust platform for evaluating trading strategies using historical data. It enables users to simulate trades and analyze their performance under various market conditions.

## Features
- **Historical Data Analysis**: Analyze your trading strategies with a wide range of historical market data.
- **Strategy Simulation**: Simulate trades based on user-defined strategies to evaluate their effectiveness.
- **Performance Metrics**: Access various metrics to assess strategy performance, including return on investment (ROI) and maximum drawdown.
